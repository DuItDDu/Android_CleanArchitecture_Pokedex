<p align="center">  
This app is a clean architecture example with the theme of Pokedex.
It was developmented using Jetpack compose, Hilt, Coroutines, Navigation, Paging, etc.
(Refer to https://github.com/ozantas/CleanPokedex)
</p>

## Tech Stack & Features
- Kotlin
- Clean architecture
- Jetpack Compose
- Coroutines
- Hilt
- Navigation
- Retrofit
- Paging3
- Support dark theme
